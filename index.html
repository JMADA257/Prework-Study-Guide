GIT:

  • GIT clone - creates a repository of the branch onto my local device.
  • GIT checkout - moves from the working branch to a new one.
  • GIT checkout -b - creates a new branch.
  • GIT checkout -b branch-name - creates a new branch and switches to it.
  • This is in order to push your feature branch to the main on Github.
  • GIT add . - adds modifications in the current branch to the staging area, A indicates to add all changes.
  • GIT commit -m "type your comment here" - captures a snapshot of the current staged changes. Along with a description.
  • GIT pull origin main - pulls the most recent branch into the local drive, origi nindicates this comes from github repo, main indicates the branch.
  • GIT push origin 'branch name" - pushes the code to the origin (github) followed by a branch name.

JavaScript:

  • let - Declares a vaiable.
  • let myVariable; - 'myvariable' is the variable, while the ; indicates wwhere the statement ends.
  • JavaScript is case sensative.
  • let my variable = "bob"; - '"bob"' is the value of the variable
  • var - Declares a variable.
  • string - to signify that the value is a string enclose it in quotes or double quote marks.
  • Number - to signify that the value is a number, it does not have to be in quotes.
  • Boolean - this is a true or false value with no need of quotes.
  • Array - lets you structure multiples values in a single reference ex: let myvariable = [1, 'Bob', 'Steve',10].
  • Object - this can be anything, everything in javascript is an object and can be stored in a variable. EX let myVariable = document.querySelector('h1');
  • Comments - snippets of text that can be added with code to help explain, the browser will ignore these.
  • to make a comment -
  • /*
  • Everything in between these is a comment
  • */
  • OR
  • // This is a comment as well
  • Operators - A mathematical symnbol that produces a result based on 2 values.
  • + - add two numbers together or combine two strings.
  • -, *, / - subtract multiply and divide, basic math symbols.
  • === - this performs a test to see if the two values are equal, will result in a true/false (boolean) result.
  • !, !== - this turns the value into the opposite EX if its a true it will turn into a false. As well if you use this with the equality operator it will test to see if the values are not equal.
  • Primitive values -
  • Null - return value = "object"
  • Undefined - return value = "undefined"
  • Boolean - Return value = "boolean
  • Number - Return value = "number"
  • Bigint - return value = "bigint"
  • String - return value = "string"
  • Symbol - return value = "symbol"
